* When you first run the application, there is only one User in the database: ADMIN
* You need to create Users and assign their roles.
* After creating Users, provide them with a login and password so that they can start using the application
* Remember that when creating a new User, the ADMIN role is assigned to him by default.
* Accidentally setting the role to ADMIN/MANAGER makes confidential information available to an unauthorized person.
* Pay special attention to assigning the appropriate Role to the Users.
